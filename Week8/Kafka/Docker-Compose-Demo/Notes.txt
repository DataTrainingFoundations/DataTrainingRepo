1. podman compose up -d --build
2. podman ps
3. podman exec -it generator bash
4. python event_generator.py
5. (in another shell) podman exec -it spark-master bash
6. cd /app
7. spark-submit \
  --master spark://spark-master:7077 \
  --packages org.apache.spark:spark-sql-kafka-0-10_2.12:3.5.6 \
  /app/spark_kafka_consumer.py
8. in both shells type: ctrl+d 
9. podman compose down -v

Note: older Kafka version than when we ran Kafka locally, also, use docker in place of podman
if you are using docker
