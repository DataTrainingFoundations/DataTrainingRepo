# Job Description (Training Aligned)

Candidates have been trained to:

- Develop Python programs using control flow, functions, modules, and object-oriented programming
- Work with Python data structures and standard libraries
- Process and analyze data using NumPy, Pandas, and Matplotlib
- Perform file handling, exception handling, logging, and testing
- Write SQL queries using joins, subqueries, aggregate functions, and CRUD operations
- Understand relational database design, normalization, and constraints
- Understand Big Data and Data Warehousing fundamentals
- Work with Apache Spark (RDDs, Spark SQL, and DataFrames)
- Understand Spark cluster architecture and distributed job execution
- Use Git for source control and collaboration
- Demonstrate awareness of Agile development practices, Cloud Computing, and Generative AI concepts

# Coding Exercise Scope

Panels may include a small coding exercise within training scope, such as:

- Python programming using loops, functions, or data structures
- File handling or exception handling tasks
- Basic Pandas data manipulation
- SQL queries using joins or aggregate functions

The focus should remain on **problem-solving ability and programming fundamentals**.
